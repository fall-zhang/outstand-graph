# Outstand Graph

图表低代码解决方案

- 面向所有人：生成图表，绑定数据，简单交互
- 面向程序员：简化图表编辑流程，不再对属性迷惘

> 本应用使用该模板搭建而成：https://github.com/fall-zhang/vite-vue3-TS-lint

## 项目启动

- `pnpm i` 安装依赖
- `pnpm dev` & `pnpm serve` 启动项目

## 项目内容

### 图表编辑

- 实现导入导出数据图
- 实现自由配置 echarts 图表样式

### 图表生成（开发中）

- 实现导入表格，选中，确定渲染数据
- 远程数据渲染

> 更多特性和介绍施工中...
