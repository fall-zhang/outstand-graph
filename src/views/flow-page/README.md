# flow page

使用表单编辑 echarts **零门槛** 实现图表编辑

## 项目特点

- 简单的流程图工具
- 导出图片，文件，导入编辑
- 暂存配置到服务器，何人都可以编辑图表内容
- 将流程和图形绑定

### 功能

- [ ] 无论何时，长按空格都可以进行拖拽整个画布（关闭选择和多选）
- [ ] 无论何时，点击 ctrl 都切换为指针可以进行多选

- [ ] 迷你地图
- [ ] 顶部菜单或者是左侧菜单
